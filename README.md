# React + TypeScript + Vite

Data provided by STRATZ. Request your API token [here](https://stratz.com/api), and enter it in server/vars.env

## Setup

pip install -r requirements.txt
npm install
python serever/data.py

## Running

python server/app.py
npm run dev
